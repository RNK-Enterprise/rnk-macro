# RNK™ Macro Button

A free Foundry VTT module by RNK Enterprise.

Injects a configurable macro execution button directly into every actor sheet header — system-agnostic, zero dependencies.

## Features

- Adds a button to every actor sheet header (works on all game systems)
- Left-click the button to execute the assigned macro
- Right-click the button to open the macro picker and assign any world macro
- Macro assignment is stored per-actor — each character remembers its own macro
- Button updates in real-time after assignment
- Zero configuration required — install and use

---

## Installation

Paste this manifest URL into Foundry's module installer:

```
https://github.com/RNK-Enterprise/rnk-macro/releases/latest/download/module.json
```

---

## Usage

1. Open any actor sheet
2. Locate the RNK Macro Button in the sheet header
3. **Right-click** the button to open the macro picker
4. Select any macro from your world macro list
5. **Left-click** the button at any time to execute the assigned macro

To remove an assigned macro: right-click the button and click "Remove Assignment".
